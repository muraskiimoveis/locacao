Explicações para os códigos dos vídeos
 
1. Youtube
Para pegar o o código de um vídeo do youtube.com para inserir no campo "Código Vídeo" no sistema, copie do endereço URL, somente o código circulado em vermelho conforme a figura abaixo (o código que deve ser selecionado fica depois do "v=").
Exemplo de código: kMcKwrtcXCQ
 
 
2. Globo
Para pegar o código de um vídeo do globo.com para inserir no campo "Código Vídeo" no sistema, copie do endereço URL, somente o código circulado em vermelho conforme a figura abaixo (o código que deve ser selecionado fica depois da palavra "GIM" e antes do "-" (hífen)).
Exemplo de código: 1042465
 
 
3. Blip
Para pegar o o código de um vídeo do blip.tv para inserir no campo "Código Vídeo" no sistema, deve seguir os passos abaixo:
1° Escolha no primeiro quadro que está escrito "Share" a opção "Embed";
2° No segundo quadro escolha a opção "Show Player" e clique no botão "GO";
3° Pegue somente o código que está circulado em vermelho do 3° passo (o código que deve ser selecionado fica depois do "play/" e antes da " (aspa)).
Exemplo de código: AYGE1lEA
